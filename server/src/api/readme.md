# API 디렉터리 구조 설명

| 파일 이름 | 설명 |
| ---- | ------ |
| routes | 라우터를 담당합니다. |
| controller | 비즈니스 로직을 담당합니다. |
| sql | sql 쿼리를 수행하는 함수를 담당합니다. |
| validate | 폼 데이터를 검증하거나, 비즈니스 로직을 수행하기 전 점검사항에 대한 함수를 담당합니다. |
| types | 타입스크립트의 타입 관련 인터페이스, 타입, 함수를 담당합니다. |
| service | 컨트롤러에서 로직이 너무 클 경우 로직 분리를 담당합니다. |
